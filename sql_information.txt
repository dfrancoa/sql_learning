SQL information

Example, table creation and manipulation:

Create table:

CREATE TABLE friends (
   id INTEGER,
   name TEXT,
   birthday DATE
);


Add into row information:

INSERT INTO friends (id, name, birthday) 
VALUES (1, 'Ororo Munroe', '1940-05-30');

INSERT INTO friends (id, name, birthday) 
VALUES (2, 'Ivan', '1982-09-09');

INSERT INTO friends (id, name, birthday) 
VALUES (3, 'Edder', '1985-04-05');


Updating a row:

UPDATE friends
SET name = 'Storm'
WHERE id = 1;


Adding a column into table:

ALTER TABLE friends 
ADD COLUMN email TEXT; 


Updating row:
